## 🍎 MacOS 安装教程（Codex.dmg）

Mac 用户目前无法通过国区 App Store 下载，需要使用官方安装包。

### 安装步骤：

1. 打开上方 **Codex.dmg 下载链接**
2. 等待安装包下载完成
3. 双击打开 `Codex.dmg`
4. 将 **Codex 应用拖入 Applications（应用程序）文件夹**
5. 首次打开时，根据系统提示点击“允许运行”

> 💡 如果提示“无法打开”，请前往：  
> 系统设置 → 隐私与安全 → 允许该应用运行
